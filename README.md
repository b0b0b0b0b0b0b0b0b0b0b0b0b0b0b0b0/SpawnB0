# SpawnB0

Плагин для установки точки спавна с системой кулдаунов и красивыми hex-сообщениями.

## Возможности

- Установка точки спавна командой `/setspawn`
- Телепортация на спавн командой `/spawn`
- Система кулдаунов между использованиями
- Обратный отсчет с красивым отображением каждую секунду
- Отмена телепортации при движении
- Поддержка hex-цветов в сообщениях
- Мультиязычность (русский/английский)

## Установка

1. Скачайте `.jar` файл из папки `target`
2. Поместите его в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Настройте конфигурацию в `plugins/SpawnB0/config.yml`

## Команды

- `/setspawn` - Установить точку спавна (требует права `spawnb0.setspawn`)
- `/spawn` - Телепортироваться на спавн (требует права `spawnb0.spawn`)

## Права доступа

- `spawnb0.setspawn` - Позволяет устанавливать точку спавна (по умолчанию: OP)
- `spawnb0.spawn` - Позволяет телепортироваться на спавн (по умолчанию: все)

## Конфигурация

Файл `config.yml`:

```yaml
spawn:
  cooldown: 10              # Кулдаун между использованиями (в секундах)
  teleport-delay: 5          # Задержка перед телепортацией (в секундах)
  cancel-on-move: true       # Отменять телепортацию при движении
```

## Языковые файлы

Плагин поддерживает два языка:
- `lang_ru.yml` - Русский
- `lang_en.yml` - Английский

Все сообщения настраиваются через эти файлы с поддержкой hex-цветов в формате MiniMessage.
